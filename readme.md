# PetitVideoEditor

A small video editor for creating video montages from local video clips.

## Dev instructions

### Installation

To install the development environment, just run:

```bash
yarn install
```

To export the website, run:

```bash
yarn build
```

To run the dev server, run:

```bash
yarn dev
```
